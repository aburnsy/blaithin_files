# Blaithin

Plant price comparison across Irish, UK, and EU online nurseries — built
from daily scrapes against [data sources](nurseries) matched to the [RHS
plant database](https://rhs.org.uk/plants).

## Browse

- [**Plants**](plants) — search by sun/soil/hardiness, sort by price across nurseries
- [**Cultivar detail**](cultivar) — one row per nursery for a specific cultivar, with delivery + total to door
- [**Non-plants**](non-plants) — compost, tools, pots, fertilisers
- [**Nurseries**](nurseries) — per-source overview (currency, delivery, VAT)
- [**Health**](health) — scrape run history
